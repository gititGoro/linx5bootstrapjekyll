CreateAccount
============

Allows you to create individual accounts in a Xero organisation.

Properties
----------

-  #### Authentication
[Authentication](../../../Common/Authentication/Index.md) values used to connect to the Xero server.
-  #### Account
Account to create.


Output
-----
#### Account
The account created.
