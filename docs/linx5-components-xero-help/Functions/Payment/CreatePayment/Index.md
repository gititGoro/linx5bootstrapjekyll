CreatePayment
============

Use this function to apply payments to approved AR and AP invoices or refund AR or AP credit notes.

Properties
----------

- #### Authentication
[Authentication](../../../Common/Authentication/Index.md) values used to connect to the Xero server.
- #### Payment
Payment to create.


Output
-----
#### Payment
The payment created.
