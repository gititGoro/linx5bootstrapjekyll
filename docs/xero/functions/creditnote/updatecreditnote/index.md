UpdateCreditNote
============

Allows you to update draft credit notes.

Properties
----------

-  #### Authentication
[Authentication](../../../Common/Authentication/Index.md) values used to connect to the Xero server.
-  #### Credit note Id
Unique identifier of the credit note to update.
-  #### Credit note
Credit note to update.


Output
-----
#### CreditNote
Credit note updated.
