CreateContactGroup
============

Use this function to create contact groups.

Properties
----------

-  #### Authentication
[Authentication](../../../Common/Authentication/Index.md) values used to connect to the Xero server.
-  #### Contact group
Contact group to create.


Output
-----
#### ContactGroup
The contact group created.
