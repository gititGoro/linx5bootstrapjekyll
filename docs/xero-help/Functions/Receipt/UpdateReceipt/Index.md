UpdateReceipt
============

Use this function to update DRAFT receipts.

Properties
----------

- #### Authentication
[Authentication](../../../Common/Authentication/Index.md) values used to connect to the Xero server.
- #### Receipt Id
Unique identifier of the receipt to update.
- #### Receipt
Receipt to update.


Output
-----
#### Receipt
Receipt updated.
