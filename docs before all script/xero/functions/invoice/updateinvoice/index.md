UpdateInvoice
============

Use this method to update an invoice record.

Properties
----------

- #### Authentication
[Authentication](../../../Common/Authentication/Index.md) values used to connect to the Xero server.
- #### Invoice id / number
Unique identifier of the invoice to update.
- #### Invoice
Invoice to update.


Output
-----
#### Invoice
Invoice updated.
