Status
======

Show changed files in a repository.

Properties
----------

- #### Repository
Path to a local repository.

Output
------
A list of files with their status code.  The codes used to show the status of files are:

- M = modified
- A = added
- R = removed
- C = clean
- ! = missing 
- ? = not tracked
- I = ignored
